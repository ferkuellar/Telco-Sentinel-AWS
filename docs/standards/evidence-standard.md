# Evidence Standard

## Objective

Ensure every phase is reproducible, reviewable, and auditable.

## Required Evidence Per Phase

- screenshots/
- txt/
- notes/
- validation-checklist.md

## Minimum CLI Outputs

- terraform init
- terraform validate
- terraform plan
- cloud service verification commands
- git status

## Screenshot Rules

- use meaningful names
- capture console evidence clearly
- avoid cropped screenshots that hide key metadata

## Naming Convention

- F0-Tree.png
- F0-Versions.png
- 01-terraform-init.txt
- 02-tool-versions.txt
- 03-git-status.txt
